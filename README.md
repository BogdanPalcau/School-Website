# School Website (Rangoon International Education LMS)

A full-stack Learning Management System (LMS) designed for Rangoon International Education. The platform provides a complete digital learning environment where teachers can manage courses and students can access learning materials, schedules, events, and assignments in one place.

---

## 🚀 Features

### 👨‍🏫 Teacher Features

* Create and manage courses
* Upload learning materials (PDF, DOCX, PPTX, etc.)
* Schedule classes and events
* Create assignments and deadlines
* Manage student submissions

### 🎓 Student Features

* Enroll in courses
* View class timetable and events
* Access course materials
* View documents directly in-browser
* Submit assignments

---

## 🔄 Core System Design

* Real-time synchronization between teacher updates and student dashboards
* Unified calendar for classes, events, and deadlines
* Role-based access control (student / teacher / admin)

---

## 🛠️ Tech Stack

* PHP (backend)
* HTML / CSS / JavaScript (frontend)
* SQLite/MySQL (database)
* XAMPP local development environment

---

## 📁 Project Structure

* `/public` – Frontend pages
* `/uploads` – Course files and submissions
* `/database` – Database storage
* Core PHP files handle authentication, routing, and logic

---

## 🎯 Goal

To build a modern, intuitive LMS similar to Blackboard but more lightweight, faster, and easier to use for both students and teachers.

---

## 📌 Status

Work in progress – core system implemented, additional features and refinements ongoing.
